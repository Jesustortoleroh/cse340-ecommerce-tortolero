# E-commerce Site with Shopping Cart — CSE 340

This is a server-side E-commerce web application built for CSE 340.  
It includes features such as user authentication, product management, shopping cart, checkout workflow, and an admin dashboard.

## Tech Stack
- Node.js with Express.js as the backend framework
- EJS or Liquid.js for rendering views
- ESM (ECMAScript Modules)
- PostgreSQL for the database
- Express-session authentication
- Deployed on Render with a connected PostgreSQL database

## Features
- Product catalog
- Product detail pages
- Shopping cart
- Checkout workflow (multi-stage process)
- User accounts + roles (admin, staff, customer)
- Admin dashboard for product and content management
